# Rendering

- Static rendering: render tĩnh là dữ liệu được tạo ra tại thời điểm build và đc lưu duowics dạng file HTML. Sẽ được server trả về ngay khi có request. Tốc độ nhanh, nhưng không phù hợp với nội dung thay đổi thường xuyên.
- Dynamic rendering: render động là dữ liệu được tạo ra tại thời điểm request. Server sẽ xử lý logic và tạo ra HTML mới mỗi khi có request. Phù hợp với nội dung thay đổi thường xuyên, nhưng có thể chậm hơn so với static rendering.
